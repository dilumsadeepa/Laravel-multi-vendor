# Laravel-multi-vendor

How to install

1) clone the repo
2) In project folder open the Terminal or cmd
3) In Terminal type

    composer install
    
4) if composer install doesn't work
    composer update && composer install
    
5) then

    npm install
    npm run dev
    npm run build
    
    php artisan key:generate
    
    php artisan migrate
    
    php artisan serve
    
6) Then open bowser and goto localhost:8000
